Branded code emails with a 3D item orbit.

Full branded HTML in the site's dark + gold palette — mono eyebrow, gold target mark, your real finds spinning in a faux-3D orbit, and the big gold code. Copy each field straight into Supabase. {{ .Token }} renders as the 6-digit code.

Two prerequisites for this to render right

1) The SMTP Sender email must be on a Resend-verified domain (e.g. noreply@findersxkeepers.com), not @gmail.com. 2) The item images load from https://www.findersxkeepers.com/brand/finds/…, so they appear once the site is deployed. Until then the orbit shows its colored gradient tiles — which is also the fallback when a mail client blocks images.

Where each field goes

  • 1Supabase → Authentication → Email Templates → open Magic Link. Returning users get a sign-in code here.
  • 2Replace Subject heading and Message body with the copied fields. Clear whatever's there first — paste the whole document.
  • 3Repeat for Confirm signup (first-time emails).
  • 4Save both, then resend a code from /login.
The orbit spins/floats in Apple Mail & iOS Mail (they honor <style> keyframes). Gmail & Outlook strip animation and show the same tiles, just static — so it looks intentional everywhere.
